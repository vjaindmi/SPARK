3.   Функции Приложения One-Key
Приложение One-Key предлагает функции для настройки Инструмента (см. п. 3.2), для инвентаризации (см. п. 3.3) и для создания отчетов Инструмента (см. оговорку 3.4) в сочетании с определенными Инструментами и принадлежностями Milwaukee, «One-Key ready» («Инструмент One Key»).
Приложение One-Key позволяет выполнять настройку определенных параметров Инструментов One-Key, например, скорость и момент вращения сверлильных станков. Устройство передает управляющие команды на Инструмент One-Key посредством беспроводного соединения Bluetooth. Пожалуйста, узнайте в руководстве для своего Инструмента, можно ли и в каком объеме этот Инструмент можно использовать с Приложением One-Key. Функцию настройки Инструмента из Приложения One-Key можно использовать изолированно, то есть без применения системы инвентаризации или отчетов Инструмента.
Приложение One-Key включает функцию инвентаризации, позволяющую (i) создавать инвентарные списки Инструментов One-Key и Инструментов третьих лиц, и (ii) собирать и использовать информацию о применении этих Инструментов и составлять соответствующие отчеты. Эти функции предоставляются Приложением One-Key, однако отдельные функции при необходимости можно использовать только вместе с онлайн-платформой для управления инвентаризацией (веб-сайт) («Платформа для управления инвентаризацией»). При использовании функции инвентаризации Приложение One-Key автоматически через соединение Bluetooth собирает и сохраняет определенные данные зарегистрированных Инструментов One-Key (например, серийный номер, архив использования) и через интернет-соединение Устройства (при его наличии) передает эти данные на Сервер Платформы для управления инвентаризацией («Сервер One-Key»).
Этот Договор не включает использование Платформы для управления инвентаризацией (веб-сайт). По вопросам общих условий заключения сделки в отношении Платформы для управления инвентаризацией см. http://ru.milwaukeetool.eu/one-key/terms-and-conditions/. Мы оставляем за собой право в любое время прервать соединение Приложения One-Key с Сервером One-Key и не обязуемся обеспечивать доступность Сервера One-Key в конкретный момент времени.
Приложение One-Key может собирать информацию об Инструментах One-Key и их использовании («Данные Инструмента») и составлять соответствующие отчеты (при необходимости в сочетании с Платформой для управления инвентаризацией), а также давать указания о рекомендуемом техобслуживании.
Чтобы пользоваться Приложением One-Key, необходимо в режиме онлайн создать Учетную запись («Учетная запись One-Key»). Для этой регистрации требуется интернет-соединение, ее можно выполнить из Приложения One-Key. Для использования Приложения One-Key вместе с отдельным Инструментом One-Key необходимо связать Инструмент One-Key (его ID) с Вашей Учетной записью One-Key.
